# Launch kit

## One-sentence description

Ableton Note Helper is a free browser-based reference for exploring scales, modes, chords, arpeggios and exact voicings on iPad, iPhone and Push pad grids.

## Short description

Choose a device layout and root, compare several patterns at once, or tap pads to identify chords and inversions. The app is open source, dependency-free, requires no account and performs no tracking.

## Suggested announcement

I wanted accurate practice diagrams for Ableton Note, but static charts became crowded and often obscured the actual grid geometry. I built a free interactive reference that generates the iPad, iPhone and Push layouts algorithmically.

It supports multiple simultaneous scales, modes, chords, arpeggios and exact voicings, plus chord recognition from pads selected by the user. It runs entirely in the browser, requires no account and is open source.

Corrections to device geometry, theory data and chord naming are welcome.
